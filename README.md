# device_detact

一個用於比較不同 User Agent 解析器效果的網頁工具。

## 專案簡介

這是一個設備檢測工具，主要用於：

- 比較不同 UA Parser 的解析結果
- 測試和分析 User Agent 字串
- 展示各種設備和瀏覽器的識別能力
