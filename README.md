# device_detact

![Version](https://img.shields.io/badge/version-v1.1.0-blue)

一個用於比較不同 User Agent 解析器效果的網頁工具。

## 專案簡介

這是一個設備檢測工具，主要用於：

- 比較不同 UA Parser 的解析結果
- 測試和分析 User Agent 字串
- 展示各種設備和瀏覽器的識別能力

## 線上展示

可以透過 GitHub Pages 查看：[Device Compare Demo](https://alexsuze.github.io/device_detact/)
